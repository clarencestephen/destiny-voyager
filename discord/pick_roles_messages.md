# Reaction-role messages — paste these into `#recruitment-roles`

These are the messages to post in `#recruitment-roles` after the server structure is built. Modelled after the Badass Clan setup. Use MEE6 or Sapphire to wire up the reactions → roles after posting.

Mods reach the role config through:
- **MEE6**: https://mee6.xyz/dashboard → your server → Reaction Roles → New
- **Sapphire**: https://sapphirebot.dev/ → Reaction Roles plugin → Add

Each message below is one block to send (one per Discord message so the bot can register reactions separately).

---

## Message 1 — Welcome (pinned)

```
:wave:  Welcome to Order 66 — the dark side of Destiny 2 build optimization. :wave:

**Start here.** Read 📋 #imperial-law → react to messages below to pick your roles → introduce yourself in #in-a-galaxy-far-far-away.

We don't tag @everyone or @here, so you need to pick what applies. Otherwise you won't see that channel or get tagged in it. We try not to blow up phones 24/7.

If you're looking for a clan, we're fun and active. Weekly events. No guardian left behind.

P.S. If you're not signed up with @Charlemagne (the Discord D2 bot), you'll need to be signed in to join events.
```

## After posting

1. Open MEE6 (or Sapphire) dashboard
2. Find each message in `#recruitment-roles` by ID
3. Map emoji → role per the tables above
4. Test by reacting on your own — should toggle the role on you
5. Pin Messages 1-8 in `#recruitment-roles` (right-click → Pin Message)

Once verified, set the channel's "Send Messages" permission to **deny @everyone** so only the reaction-role messages stay there. That way nobody clutters the channel.
